üåà CSS BORDER, PADDING & MARGIN ‚Äî ONE PAGE NOTE

üß± BORDER  
Defines the outline around an element.  
Longhand:
border-top: 5px solid red;
border-right: 5px dashed green;
border-bottom: 5px double blue;
border-left: 5px dotted orange;

Each side can have: border-*-width, border-*-style, border-*-color

Shorthand:
border: 5px solid red;  /* all sides */
border-width: 5px 10px 15px 20px;   /* top | right | bottom | left */
border-style: solid dashed double dotted;
border-color: red green blue orange;

üì¶ PADDING  
Adds space inside the element (between content and border).  
Longhand:
padding-top: 10px;
padding-right: 20px;
padding-bottom: 30px;
padding-left: 40px;

Shorthand:
padding: 10px;                /* all sides */
padding: 10px 20px;           /* top-bottom | left-right */
padding: 10px 20px 30px;      /* top | left-right | bottom */
padding: 10px 20px 30px 40px; /* top | right | bottom | left */

üìè MARGIN  
Adds space outside the element (between elements).  
Longhand:
margin-top: 10px;
margin-right: 20px;
margin-bottom: 30px;
margin-left: 40px;

Shorthand:
margin: 10px;                /* all sides */
margin: 10px 20px;           /* top-bottom | left-right */
margin: 10px 20px 30px;      /* top | left-right | bottom */
margin: 10px 20px 30px 40px; /* top | right | bottom | left */

üß≠ QUICK SUMMARY TABLE
Property | Longhand Sides | Shorthand Example | Description
---------|----------------|------------------|-------------
Border   | top, right, bottom, left | border: 5px solid red; | Outline around element
Padding  | top, right, bottom, left | padding: 10px 20px 30px 40px; | Space inside element
Margin   | top, right, bottom, left | margin: 10px 20px 30px 40px; | Space outside element

‚úÖ Remember:
- padding = inner spacing
- margin = outer spacing
- border = visible outline around element
